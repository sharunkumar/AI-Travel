### Preview app locally

Clone the app:
  ```
  git clone https://github.com/sharunkumar/AI-Travel
  cd AI-Travel
  ```

rename `.env.example` to `.env` and fill in the relevant values (note: OpenAPI Key is needed for this project)

install modules and run
```
pnpm install
pnpm dev
```
